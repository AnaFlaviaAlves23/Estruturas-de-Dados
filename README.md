Objetivo: Compreender os principais conceitos de algoritmos e estruturas de dados, estudando diferentes formas de organização e armazenamento da informação. Desenvolver a capacidade de implementar, em linguagem C, as operações fundamentais dessas estruturas, analisar a eficiência e a complexidade dos algoritmos utilizados e decidir, diante de problemas reais, qual estrutura de dados é a mais adequada para uma solução correta e eficiente.
